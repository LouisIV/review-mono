# Tauri GUI with Daemon Discovery — Implementation Plan

> **For Hermes:** Use subagent-driven-development skill to implement this plan task-by-task.

**Goal:** Add a Tauri-based GUI app that auto-discovers review daemons (local + remote SSH) and provides full session management — diffs, commits, comments, approve/reject.

**Architecture:** New `gui/` directory with Tauri v2 (Rust backend) + React/TypeScript frontend. Rust backend talks to daemons via HTTP (the existing JSON API on port 7080). Daemon discovery: local = probe `localhost:7080/health`; remote = SSH tunnel + HTTP probe. Settings stored in Tauri's app data dir (`~/.config/review/gui-settings.json`). CI adds macOS aarch64 builds for the GUI.

**Tech Stack:** Tauri v2, React 19 + TypeScript, Bun (for bundling, faster than esbuild), Rust (Tauri backend with reqwest for HTTP), Tailwind CSS

**Step 1: Create daemon.rs with discovery logic**

```rust
use serde::{Deserialize, Serialize};
use std::time::Duration;

#[derive(Debug, Clone, Serialize, Deserialize)]
pub struct DaemonInfo {
    pub host: String,
    pub port: u16,
    pub version: String,
    pub build_id: String,
    pub pid: u32,
    pub source: DaemonSource,
}

#[derive(Debug, Clone, Serialize, Deserialize, PartialEq)]
pub enum DaemonSource {
    Local,
    #[allow(dead_code)]
    Remote { host: String },
}

#[derive(Debug, Deserialize)]
struct HealthResponse {
    ok: bool,
    version: String,
    build_id: String,
    pid: u32,
}

#[tauri::command]
pub async fn discover_daemons() -> Result<Vec<DaemonInfo>, String> {
    let mut daemons = Vec::new();

    // Probe local daemon
    match probe_daemon("127.0.0.1", 7080).await {
        Ok(info) => daemons.push(info),
        Err(_) => {} // no local daemon running — that's fine
    }

    Ok(daemons)
}

async fn probe_daemon(host: &str, port: u16) -> Result<DaemonInfo, String> {
    let client = reqwest::Client::builder()
        .timeout(Duration::from_secs(3))
        .build()
        .map_err(|e| e.to_string())?;

    let url = format!("http://{}:{}/health", host, port);
    let resp = client.get(&url).send().await.map_err(|e| e.to_string())?;

    let health: HealthResponse = resp.json().await.map_err(|e| e.to_string())?;

    Ok(DaemonInfo {
        host: host.to_string(),
        port,
        version: health.version,
        build_id: health.build_id,
        pid: health.pid,
        source: DaemonSource::Local,
    })
}
```

**Step 3: Extend daemon.rs with remote discovery**

```rust
use crate::settings::SshHost;

#[tauri::command]
pub async fn discover_daemons(ssh_hosts: Vec<SshHost>) -> Result<Vec<DaemonInfo>, String> {
    let mut daemons = Vec::new();

    // Local probe
    match probe_daemon("127.0.0.1", 7080).await {
        Ok(info) => daemons.push(info),
        Err(_) => {}
    }

    // Remote probes via SSH
    for host in &ssh_hosts {
        match probe_remote(host).await {
            Ok(info) => daemons.push(info),
            Err(e) => eprintln!("SSH probe {} failed: {}", host.label, e),
        }
    }

    Ok(daemons)
}

async fn probe_remote(host: &SshHost) -> Result<DaemonInfo, String> {
    let local_port = 17080u16; // fixed port for SSH tunnel, single-threaded is fine

    // Kill any existing tunnel on this port
    let _ = tokio::process::Command::new("lsof")
        .args(["-ti", &format!(":{}", local_port)])
        .output()
        .await;

    let ssh_cmd = format!(
        "ssh -f -N -L {}:127.0.0.1:{} -p {} {} -o StrictHostKeyChecking=no -o ConnectTimeout=5",
        local_port, host.daemon_port, host.ssh_port, host.hostname
    );

    let output = tokio::process::Command::new("sh")
        .arg("-c")
        .arg(&ssh_cmd)
        .output()
        .await
        .map_err(|e| format!("ssh spawn failed: {}", e))?;

    if !output.status.success() {
        return Err(format!("ssh tunnel failed: {}", String::from_utf8_lossy(&output.stderr)));
    }

    // Give tunnel a moment to establish
    tokio::time::sleep(std::time::Duration::from_millis(500)).await;

    match probe_daemon("127.0.0.1", local_port).await {
        Ok(mut info) => {
            info.source = DaemonSource::Remote {
                host: host.label.clone(),
            };
            Ok(info)
        }
        Err(e) => Err(e),
    }
}
```

```tsx
import { useState, useCallback } from "react";
import { invoke } from "@tauri-apps/api/core";

export interface SshHost {
  label: string;
  hostname: string;
  ssh_port: number;
  daemon_port: number;
}

export interface DaemonInfo {
  host: string;
  port: number;
  version: string;
  build_id: string;
  pid: number;
  source: "Local" | { Remote: { host: string } };
}

export function useDaemonDiscovery() {
  const [daemons, setDaemons] = useState<DaemonInfo[]>([]);
  const [loading, setLoading] = useState(false);
  const [error, setError] = useState<string | null>(null);

  const discover = useCallback(async (sshHosts: SshHost[]) => {
    setLoading(true);
    setError(null);
    try {
      const result = await invoke<DaemonInfo[]>("discover_daemons", { sshHosts });
      setDaemons(result);
    } catch (e) {
      setError(String(e));
    } finally {
      setLoading(false);
    }
  }, []);

  return { daemons, loading, error, discover };
}
```

**Step 4: Create src/hooks/useSettings.ts for SSH host persistence**

```tsx
import { useState, useEffect, useCallback } from "react";
import { Store } from "@tauri-apps/plugin-store";
import type { SshHost } from "./useDaemonDiscovery";

export function useSettings() {
  const [sshHosts, setSshHosts] = useState<SshHost[]>([]);
  const [store, setStore] = useState<Store | null>(null);

  useEffect(() => {
    Store.load("settings.json").then(setStore);
  }, []);

  useEffect(() => {
    if (!store) return;
    store.get<SshHost[]>("ssh_hosts").then((hosts) => {
      if (hosts) setSshHosts(hosts);
    });
  }, [store]);

  const save = useCallback(
    async (hosts: SshHost[]) => {
      if (!store) return;
      setSshHosts(hosts);
      await store.set("ssh_hosts", hosts);
      await store.save();
    },
    [store]
  );

  const addHost = useCallback(
    (host: SshHost) => save([...sshHosts, host]),
    [sshHosts, save]
  );

  const removeHost = useCallback(
    (label: string) => save(sshHosts.filter((h) => h.label !== label)),
    [sshHosts, save]
  );

  const updateHost = useCallback(
    (label: string, updated: SshHost) =>
      save(sshHosts.map((h) => (h.label === label ? updated : h))),
    [sshHosts, save]
  );

  return { sshHosts, addHost, removeHost, updateHost };
}
```

## Notes for Implementer

- The daemon API is fully defined in `docs/daemon-api.md` — use it as the contract
- Daemon serves on `localhost:7080` by default
- The health endpoint is `GET /health` — returns `{ok, version, build_id, pid}`
- SSH remote discovery: tunnel via `ssh -L local_port:127.0.0.1:daemon_port host` then probe
- Settings stored via Tauri store plugin at the app's data directory
- All Go code stays untouched except CI changes
- Use `bun` for all JS package management and building
- Tauri v2 (latest stable)
